# life inve — 라이프 인베 리서치 워크스페이스

라이프자산운용(LIFE Asset Management) 브랜드의 리서치 허브 + **데일리 자동 요약** 정적 사이트.
GitHub Pages 배포: **https://deepvaluekim-cmd.github.io/lifeinveinve/**

## 구성

| 경로 | 내용 |
|---|---|
| `index.html` | 허브 SPA (Tailwind CDN + Pretendard). 사이드바: 데일리 시황·익스퍼트콜·서브스택·실적/컨콜 요약·**데일리 유튜브** + 종목/산업/섹터/대시보드 허브 |
| `earnings/` | 실적/컨콜 요약 노트 (예: `SMTC_FQ1FY27.html`) |
| `youtube/` | 데일리 유튜브 요약 페이지 |
| `substack/` | 데일리 섭스택 요약 페이지 |
| `docs/DEV_HANDOFF_요약생성.md` | **개발자 전달 스펙** — 3대 데일리 생성기 포맷 + 데이터소스·Tier 검증 + 사이트 연동 |
| `docs/SPEC.md` | 사이트 데모 사양 |

## 3대 데일리 요약 모드

- **A. 실적/컨콜** — 티커+분기 → 고정 골격(타임라인요약→실적vs컨센→서프라이즈→가이던스→Q&A) + 시각화 + 데이터 투명성 표. 사이드바 `실적/컨콜 요약`에서 `📄 리포트` 링크.
- **B. 유튜브** — 트래킹 채널 RSS 신규감지 → 자막 요약. 사이드바 `데일리 유튜브` + `＋ 유튜브 링크 추가` 버튼.
- **C. 섭스택** — 뉴스레터 URL 묶음 → 요약. 사이드바 `데일리 서브스택`.

각 모드 상세는 `docs/DEV_HANDOFF_요약생성.md` 참조.

## 숫자 검증 원칙 (전 모드 공통)

모든 수치에 소스 Tier 태깅: **T1** SEC 8-K/10-Q(100%) > **T2** IR 가이던스(80%) > **T4** 웹 컨센(80%) > **T3** 트랜스크립트(70%) > **E** 자체산정(≤75%). 무출처 수치는 차트에 넣지 않음. 세그먼트 합=총매출 교차검증, YoY/QoQ 재계산, 노트 말미 데이터 투명성 표 필수.
