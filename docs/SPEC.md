# 글로벌 타임라인 — 데모 스펙 (공개본)

> 기준 데모: https://deepvaluekim-cmd.github.io/lifeinveinve/
> 본 문서는 데모 동작을 정리한 공개용 사양 요약입니다. (사내 세부 운영/소스 정보는 비공개)

## 개요
포털 `운용본부 › Timeline` 에 **"글로벌 타임라인" 탭**을 추가. 미국(추후 일본·대만) 종목 유니버스를 적재하고, 커버리지 종목을 누적 관리하며 데일리 리서치를 자동 수집·요약하는 리서치 워크스페이스.

## 정보구조 (보기)
- **데일리**: 데일리 시황 / 익스퍼트 콜 / 서브스택 / 실적·컨콜 요약
- **리서치**: 종목 / 산업(ETF)
- **모니터**: 섹터 대시보드 / 대시보드 허브
- **기타**: 데이터 수집 상태

## 화면 사양 (요약)
| 뷰 | 핵심 |
|----|------|
| 데일리 시황 | 시장 코멘트·지수·섹터 동향·관심종목·이슈, 일자별 누적, 작성자/반응 |
| 익스퍼트/서브스택 | 요약 카드(클릭 시 상세 펼침) + 직접 작성 |
| 실적·컨콜 요약 | 월 캘린더 + 실적 리스트 + 요약 생성 |
| 종목 | 유니버스(커버리지/미커버 구분) + 핵심지표 + 캔들차트 + 코멘트 타임라인 |
| 산업(ETF) | 종목과 동일 레이아웃 + ETF 구성종목 매핑 |
| 섹터 대시보드 | 섹터 히트맵 + 테마 카드 |
| 대시보드 허브 | 팀 대시보드 레지스트리 + 52주 신고가 트래커 |
| 데이터 수집 상태 | 소스별 수집 모니터(14일 히트맵) |

## 데이터 모델 (요지)
```
stock   { ticker, name, sector, cov, px, chg, candles[OHLC], comments[], marks[] }
etf     { ticker, name, sector, holdings[], candles, comments }
report  { date, summary, idx[], movers, watchlist[], issues[], expertCalls[], substacks[] }
earning { ticker, date, when, epsConsensus, surprise?, status, summary? }
dashboard { title, category, desc, url, author, date }
```

## 차트
- TradingView식 일봉 캔들(거래량·이동평균·최종가 축·기간 1D/1W/1M/1Y), 코멘트 마커를 캔들 시점에 매핑.

## 데이터 연동
- **시세·지수**: 시장 데이터 API 연동 (데모는 커버리지 종목 실시간 스냅샷 반영).
- **리서치 수집**: 외부 소스(구독/수집)로 전문가콜·뉴스·실적·공시를 자동 수집·요약.
- **저장**: 데이터스토어에 일자별 누적. **배포**: 개발 서버 → 점진 오픈.

## 로드맵
1. 데일리 자동화 (1차) 2. 종목 코멘트·산업 3. 섹터/허브 (태깅 누적 후)

*세부 인터랙션은 데모와 일치하도록 구현.*
